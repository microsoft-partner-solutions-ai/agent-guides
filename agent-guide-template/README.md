## [Template] Agent Guide

**Agent Name**: Clear, descriptive name

**Description**: What problem does this agent solve?

**Target Use Cases**: Specific scenarios where this agent excels

1. Navigate to copilot.microsoft.com

2. On the left pane, select "New agent"

3. Under the "Configure" tab, add the content documented in this folder.

4. Hit the "Create" button on the upper right

5. Select "Go to agent" to navigate to agent and test, or select "Share" to get share link and select who the agent should work for.
